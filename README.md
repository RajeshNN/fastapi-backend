# Initial Fastapi backend to manage user credential

This is a basic template backend project to start with if the project requires to manage user login and credentials.

This project uses fastapi library to add user login credentials. This is a basic template to start building on further applications.

### Dependencies
* AWS dynamodb
* uvicorn 0.36.0
* fastapi 0.117.1
* pydantic 2.11.9
* PyJWT 2.10.1
* Python 3.12.0
* Windows 10
